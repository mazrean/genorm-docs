Conectiong to Database

Conectiong to Database

GenORMは*sql.DB/*sql.Txを用いてクエリを実行します。このため、データベースへの接続はdatabase/sqlと同様に行えます。ただし、現在動作確認はMySQL、MariaDBでしか行えていません。

例

import (
  "database/sql"
  _ "github.com/go-sql-driver/mysql"
)

db, err := sql.Open("mysql", "user:pass@tcp(host:port)/database?parseTime=true&loc=Asia%2FTokyo&charset=utf8mb4")
